# AI Hub Project

## 简介

AI Hub 是一个为了接入多种大型语言模型而设计的服务。它旨在积累和管理各种有效的模型调用提示（prompt），并对这些大型语言模型进行持续的测试和评估。

已支持模型列表：
1. OpenAI / gpt-3.5-turbo
2. Baidu / ERNIE-Bot-turbo
3. Ali / qwen-turbo
4. Baichuan / Baichuan2-Turbo
5. Zhipu / chatGLM_turbo
